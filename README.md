# llm-to-json

Standalone TypeScript package that transforms free-form LLM agent text into typed JSON via structured extraction.

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) >= 1.0

### Local

```bash
git clone git@github.com:fanilosendrison/llm-to-json.git
cd llm-to-json
bun install
bun test
```
